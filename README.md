📚 Knowledge Vault

Structured Thinking, Not Just Note-Taking.

A modern, minimal landing page for a Personal Knowledge Vault — a tool designed to help users organize thoughts, connect ideas, and turn learning into long-term knowledge.


🚀 Features
⦁	✨ Clean and modern UI design
⦁	🧠 Focus on structured thinking (not just notes)
⦁	🔗 Concept-based knowledge connection
⦁	📌 Problem-focused sections (scattered notes, bookmarks, etc.)
⦁	📱 Fully responsive layout
⦁	🎯 Clear call-to-actions (CTA)


🧩 Sections Overview
1.	Header / Navbar
⦁	Logo
⦁	Navigation links (Features, How It Works, Why It's Different)
⦁	Early access button

2. Hero Banner
⦁	Strong headline and description
⦁	CTA buttons

3. Problem Section
⦁	Highlights common learning issues:
⦁	Scattered notes
⦁	Bookmark overload
⦁	Lack of idea connection
⦁	Forgetting knowledge

4. Solution Section
⦁	Introduces "Second Brain" concept
⦁	Key benefits with bullet points

5. Call-to-Action Section
⦁	Encourages users to join early access

6. Footer
⦁	Social links
⦁	Copyright


🛠️ Technologies Used
⦁	HTML5
⦁	CSS3
⦁	Google Fonts
⦁	Font Awesome Icons
